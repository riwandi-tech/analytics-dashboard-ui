# Analytics Dashboard UI

> A modern, responsive analytics dashboard built with pure HTML & CSS — crafted to master CSS Flexbox layout techniques from the ground up.

![Preview](preview.png)

## 📌 About The Project

This project is a comprehensive personal analytics dashboard UI built entirely with pure HTML5 and CSS3 — no frameworks, no libraries. It was created as a portfolio project to demonstrate a deep, practical understanding of CSS Flexbox by applying every core property in a real-world layout context.

## 🎯 Learning Objectives

Through this project, the following CSS Flexbox properties are explored and applied:

- `flex-direction` — controlling the main axis direction
- `justify-content` — aligning items along the main axis
- `align-items` — aligning items along the cross axis
- `flex-wrap` — allowing items to wrap into multiple lines
- `flex-grow` — defining how items grow to fill available space
- `flex-shrink` — defining how items shrink when space is limited
- `flex-basis` — setting the initial size of a flex item
- `align-self` — overriding alignment for individual items
- `order` — controlling the visual order of flex items
